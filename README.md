#### App. ToDO 2

<p>
Cette application permet de creer, de mettre à jour le status des ToDos (check/uncheck), et de les filtrer.

**À gauche du ToDo vous le supprimer, à droite sur le pannier vous le passer de check à uncheck**

</p>
